1. 开启smb服务
搜索可选功能，更多windows功能，勾选smb直通
2. 共享文件夹
选择要共享的文件夹，右键属性，共享tab，点击共享
点击共享按钮，然后把要访问的用户加入共享访问列表，并设置权限
3. 创建用户并设置权限
创建用户
net user user 123456 /add
按下 Win + R，输入 secpol.msc。
导航到：安全设置 -> 本地策略 -> 用户权限分配。
找到 “网络访问: 共享和安全模型用于本地帐户”。
务必将其设置为“经典 - 对本地用户进行身份验证，不改变其本来身份”。
4. 开启Guest访问(可选)
启用Guest
启动组策略（gpedit.msc）→依次打开Windows设置——安全设置——本地策略——安全选项。
在列表找到来宾账户状态，双击进入→已启用→确定
给予Guest网络权限
启动组策略（gpedit.msc）→依次打开Windows设置——安全设置——本地策略——用户权限分配。
找到从网络访问这台计算机，双击进入。
找到添加用户或组，并将您的用户名添加到其中。
找到拒绝从网络访问这台计算机，双击进入→删除Guest→确定。
5. 开启匿名访问(可选)
给予匿名网络权限
启动组策略（gpedit.msc）→依次打开Windows设置——安全设置——本地策略——安全选项。
找到网络访问: 将 Everyone 权限应用于匿名用户，双击进入→已启用→确定。
找到网络访问: 本地帐户的共享和安全模型，双击进入→仅来宾→确定。
